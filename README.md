# AI-Based Missile Detection and Evacuation System 🚀

An advanced AI-powered system for real-time missile detection, tracking, threat analysis, and evacuation guidance.

## Features
- Real-time missile detection
- Object tracking using Kalman Filter
- Emergency alerts via SMS/Call
- Firebase cloud integration
- React monitoring dashboard
- AR evacuation navigation

## Tech Stack
- Python
- OpenCV
- YOLOv8
- Flask
- Firebase
- React.js
- Unity
- MongoDB
